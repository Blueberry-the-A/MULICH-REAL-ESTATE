# 📌 Практичне завдання: Верстка макету з Figma

## 🎯 Мета

Зверстати веб-сторінку відповідно до наданого макету у Figma з максимальною точністю (pixel-perfect) та дотриманням сучасних стандартів frontend-розробки.

---

## 📁 Вхідні дані

- Посилання на макет у Figma: https://www.figma.com/community/file/1201362801525377074/real-estate-landing-page-freebies
- Усі необхідні шрифти, іконки та зображення (експортувати з макету)

---

## ⚙️ Вимоги до виконання

### 1. HTML

- Семантична верстка (`header`, `main`, `section`, `footer` тощо)
- Валідний HTML5
- Логічна структура документа

### 2. CSS / SCSS

- Використання сучасних підходів (Flexbox)
- Адаптивність (mobile-first)
- БЕМ або інша зрозуміла методологія іменування класів
- Мінімізація дублювання стилів

### 3. Адаптивність

Обов’язково реалізувати:

- Mobile (320px+)
- Tablet (768px+)
- Desktop (1024px+ або згідно макету)

### 4. Інтерактивність

- Hover / focus ефекти згідно макету
- Кнопки, посилання — інтерактивні
- (опційно) прості анімації

### 5. Pixel-perfect

- Максимальна відповідність макету
- Відступи, кольори, шрифти — як у Figma

---

## ⭐ Обов'язково

- Використання SCSS / препроцесорів
- Оптимізація зображень (WebP, lazy loading)
- Додавання favicon
- Коректне підключення Google Fonts
